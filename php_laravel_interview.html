Q7 What is a Closure / Anonymous Function? ▾
Answer
A closure is a function without a name. Used for callbacks, array functions, and more.
$greet = function($name) {
    echo "Hello, {$name}";
};

$greet('Azad'); // Hello, Azad

// Using 'use' to capture outer variables
$prefix = "Mr.";
$greet = function($name) use ($prefix) {
    echo "{$prefix} {$name}";
};

Q9 What is type juggling and type casting? ▾
Answer
PHP automatically converts types (type juggling). You can manually cast types.

Q10 What is the null coalescing operator ?? ▾
Answer
Returns the left operand if it exists and is not null; otherwise returns the right.
$name = $_GET['name'] ?? 'Guest';
// Old way: isset($_GET['name']) ? $_GET['name'] : 'Guest'

// Chaining
$val = $a ?? $b ?? 'default';

Q19 What are PHP Generators? ▾
Answer
Generators allow iteration without building the full array in memory. Uses yield.
function xrange($start, $end) {
    for ($i = $start; $i <= $end; $i++) {
        yield $i;
    }
}

foreach (xrange(1, 1000000) as $num) {
    // Memory efficient!
}
Q20 What is Dependency Injection? ▾
Answer
Passing dependencies from outside instead of creating them inside. Makes code testable and decoupled.
// Without DI (tightly coupled)
class UserService {
    private $mailer;
    public function __construct() {
        $this->mailer = new Mailer(); // ❌
    }
}

// With DI (loosely coupled)
class UserService {
    public function __construct(private Mailer $mailer) {} // ✅
}

Q25 What is the N+1 problem and how to fix it? ▾
Answer
N+1 occurs when 1 query loads a collection, then N additional queries load each related model.
// ❌ N+1 Problem
$users = User::all(); // 1 query
foreach ($users as $user) {
    echo $user->posts; // N queries!
}

// ✅ Eager loading fix
$users = User::with('posts')->get(); // 2 queries
💡 Tip: Use Laravel Telescope or Debugbar to detect N+1 queries.

Q38 What is the Repository Pattern in Laravel? ▾
Answer
Adds an abstraction layer between controller and model. Makes code testable and swappable.
// Interface
interface UserRepositoryInterface {
    public function findById(int $id): User;
}

// Implementation
class UserRepository implements UserRepositoryInterface {
    public function findById(int $id): User {
        return User::findOrFail($id);
    }
}

// Bind in ServiceProvider
$this->app->bind(UserRepositoryInterface::class, UserRepository::class);

Q45 What is Soft Deleting? ▾
Answer
Soft deletes set a deleted_at timestamp instead of actually removing the record.
use SoftDeletes;

// Migration needs
$table->softDeletes();

// Delete (sets deleted_at)
User::find(1)->delete();

// Include soft deleted
User::withTrashed()->get();

// Only soft deleted
User::onlyTrashed()->get();

// Restore
User::withTrashed()->find(1)->restore();
